## üêõ Reporting Issues

### Documentation Issues

If you find problems in documentation:

1. Create issue with label `documentation`
2. Specify which document
3. Describe the problem
4. Suggest improvement (if possible)

### Modal System Issues

If you find bugs in modal system:

1. Create issue with label `modal-system`
2. Include modal ID
3. Browser and version
4. Steps to reproduce
5. Console errors
