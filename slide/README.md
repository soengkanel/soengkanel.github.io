# Welcome to [Slidev](https://github.com/slidevjs/slidev)!

## Quick Start

To start the default slide show:

- `pnpm install`
- `pnpm dev`
- visit <http://localhost:3030>

Edit the [slides.md](./slides.md) to see the changes.

## Running Specific Presentations

To run a specific presentation from a subdirectory:

```bash
npx slidev <folder>/slides.md --open
```

### Available Presentations

| Presentation | Command |
|--------------|---------|
| **SharePoint WordPress Plugin** | `npx slidev sharepoint-wp-plugin/slides.md --open` |
