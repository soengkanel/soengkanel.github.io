Instructions
Excel File Format

Your Excel file should contain the following columns:

  • Column A: NO (Worker Number)
  • Column B: PHOTO (Worker Photo)
  • Column C: NAME (Full Name)
  • Column D: SEX (M/F)
  • Column E: DOB (Date of Birth)
  • Column F: NATIONALITY
  • Column G: BUILDING
  • Column H: POSITION
  • Column I: JOINEDDATE
  • Column J: RESIGNEDDATE
Document Images

Insert document images directly into cells:

  • Column K: Passport/ID Image
  • Column N: Work Permit Image
  • Column Q: Visa Image
Important: Images must be embedded in cells, not as floating objects.
